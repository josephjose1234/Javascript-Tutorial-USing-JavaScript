JavaScript Error Handling Statements

Statement Description Example
try Defines a code block to run (to try). If an error occurs, control jumps to the catch block. try {
  // Code to be executed
} catch (error) {
  // Code to handle errors
}
catch Defines a code block to handle any error that occurred in the associated try block. try {
  // Code to be executed
} catch (error) {
  // Code to handle errors
}
finally Defines a code block to run regardless of the result of the try and catch blocks. try {
  // Code to be executed
} catch (error) {
  // Code to handle errors
} finally {
  // Code to be executed regardless of try/catch result
}
throw Defines a custom error and throws it. Typically used inside the try block to signal an error condition. throw new Error("This is a custom error message");
